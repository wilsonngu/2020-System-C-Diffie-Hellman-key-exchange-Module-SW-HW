# 2020-System-C-Diffie-Hellman-key-exchange-Module-SW-HW

A software implementation of the “Diffie-Hellman” key exchange, this allows two users to exchange a unique key over a public environment that may be insecure. The goal of this project includes hardware module implementation, utilize a hardware and software handshaking protocol, and verify the correct functionality of the hardware to software integration in System C.
